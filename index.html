Program Core

企業見学で終わらせない。
でも、重くしすぎない。

成功条件は、教育価値と事業価値を両立しながら、学校・企業・売場の負担を増やさないことです。本LPでは「運用できる探究」に必要な粒度までプログラムを設計しています。

🏫

学校は授業化しやすい

総合的な探究、商業、国際理解、キャリア教育へ接続。ワークシート・評価ルーブリック・発表テンプレート付きで教員準備を圧縮します。

🏬

企業は負担を限定できる

毎回の個別伴走や講師派遣を前提にせず、標準動画・資料提供・集約レビューで運用。生徒の企画はJTB側で整理して企業が見やすい形にします。

📊

マーケティング価値が残る

Z世代の購買視点、SNS企画、地域食材アイデア、売場体験の気づきを、企業側が使えるインサイトレポートとして返します。

Lecturer Option

講師は標準必須ではなく、必要な学校だけ選べる

多くの学校が導入できるよう、基本運用はオンライン支援＋教員進行で成立。講師登壇は学年規模、先生の経験、学校の希望に応じて追加します。

旅前導入、企画レビュー、旅後発表講評など、必要な場面だけオンライン講師を追加できます。

  • 1コマ単位で追加しやすい
  • 遠方校でも導入可能
  • 交通費不要
  • 学校側の不安を軽減
別途実費
🚄

対面講師登壇

現地導入授業、成果発表会、企業合同ワークショップなど、高付加価値化したい場合に選択します。

  • 対面講師費は別途見積
  • 交通費・宿泊費等は別途実費
  • 学校行事・成果発表会に適合
  • 実施地域・日程により調整
価格設計の明確化：標準プログラムはオンライン支援を基本にし、対面講師の交通費を標準価格に内包しません。これにより、遠方校・少人数校・初年度モニター校でも導入しやすくします。

Learning Flow

旅前・旅中・旅後を一本の学びにする

旅の中の90〜150分を、授業前後の探究活動とつなぎます。見学で終わらず、提案・評価・振り返りまで完結します。

1

タビマエ｜学校での事前学習

ブランド・農業・市場・顧客を学び、チームごとに仮説と企画案をつくります。

  • 最小：2コマ／標準：4〜6コマ
  • 成果物：企画書、POP案、SNS企画、質問リスト
  • 企業資料は標準パッケージで配布
2

タビナカ｜東京・横浜での体験学習

三越伊勢丹関連売場、カムジャバト店舗・POP UP、横浜エリアを観察し、顧客体験を分析します。

  • 現地所要：90〜150分
  • 標準人数：20〜80名／最大120名目安
  • 売場観察・顧客導線・接客・陳列・Q&A
3

タビアト｜改善・最終提案

現地で得た気づきを整理し、企画をブラッシュアップ。企業・学校関係者への最終プレゼンにつなげます。

  • 最小：1コマ／標準：2〜3コマ
  • 優秀案をJTB側で集約
  • 企業レビュー・表彰・実装検討へ接続

Risk Control

安全・権利・運営ルールを先に整える

食品、撮影、SNS、売場混雑、商標・権利を曖昧にしないことで、学校も企業も安心して参加できます。

✓
食品・アレルギー対応試食は必須にせず、観察中心でも成立。試食する場合は事前告知・代替案・保護者確認を用意します。
✓
SNS・撮影許諾撮影可否、人物映り込み、企業ロゴ、投稿範囲を事前に整理。原則は校内発表・非公開利用から開始します。
✓
売場負担の抑制混雑時間帯を避け、班別ローテーション、質問数制限、集合場所指定で通常営業への影響を抑えます。
✓
商品化・権利関係生徒アイデアの実装、名称掲載、クレジット表記、販売利用は、事前合意・監修・契約の範囲内で扱います。
