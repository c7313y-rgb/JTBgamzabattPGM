# カムジャバト × 三越伊勢丹 × JTB 修学旅行探究プログラム LP v3

## 更新ポイント
- 学校運用面を強化：最小2コマ＋旅後1コマ、標準6〜9コマ、実装検討10コマ〜の3段階に整理。
- 企業負担軽減を明確化：標準動画・資料提供・現地受入枠・月次レビューで運用できる設計。
- 講師支援をオプション制に変更：標準はオンライン支援＋教員進行。オンライン講師・対面講師は必要時のみ追加選択。
- 対面講師の費用条件を明記：対面時は講師費とは別に交通費・宿泊費等を別途実費として扱う設計。
- 企業価値を強化：Z世代インサイト、商品/売場/SNS/地域共創アイデアを成果物として残す設計。
- LP機能を追加：人数・授業枠・企業関与余力・講師サポートを入れる導入シミュレーターを実装。
- リアル感の強化：既存パンフレット素材からWebP画像を追加し、現地・売場・発表のイメージをLP内に反映。
